# TES — Tracing Eastern Sicily’s Networks

TES is a Digital Humanities research project investigating social, economic, and cultural networks in southeastern Sicily between the 10th and 5th centuries BCE.

The project combines archaeology, history, epigraphy, and spatial analysis to reconstruct regional systems of interaction across the Archaic Mediterranean.

## Research Infrastructure

TES develops its datasets using **Cadmus**, the scholarly data management framework developed by Daniele Fusi.

The infrastructure currently exposes:

- TES Editor — collaborative data creation environment  
- TES API — structured research data services

Editor:
https://tes-app.ancientsocietiesinspace.org

API:
https://tes-api.ancientsocietiesinspace.org/scalar/v1

## What this repository contains

This repository hosts the public infrastructure hub and project documentation.

It does **not** contain research data or the Cadmus backend.

---

## Collaboration

TES is conceived as an open research environment exploring ancient connectivity, landscape archaeology, and epigraphic practices.

Scholars interested in collaboration, data exchange, or methodological discussion are welcome to get in touch.
