Ancient Societies in Space

A digital research infrastructure for studying ancient societies through the integration of spatial, archaeological, and epigraphic data, enabling the analysis of networks, mobility, and landscape dynamics in the Archaic Mediterranean world.

Infrastructure

This domain hosts multiple project stacks (frontend + APIs) orchestrated via a reverse proxy. Each public endpoint is exposed through a dedicated subdomain.

  • tes-app.* for the project editor
  • tes-api.* for API services feeding the editor

Scope

The infrastructure supports digital, critical, and interpretative workflows across the ancient Mediterranean (with Sicily as a key case study), integrating spatial analysis, network perspectives, and epigraphic data.
